= README =
== About Rand'OSM ==
Rand'OSM is an Android application which goal is to provide to hikers a simple guide.

== Compatibility ==
Rand'OSM is designed to work with Android 2.3.3 (API 10) and later.

 * Android MapView Balloons - Apache Software License 2.0 - https://github.com/jgilfelt/android-mapviewballoons
 * MapView Limits (patch for Mapsforge) - JMPergar (Mapsforge dev)
